🛒 E-Commerce Product API

Welcome to the Django REST Framework powered E-Commerce API. This backend allows customers to sign up, browse, and review products, while admins can fully manage the product catalog.

📌 Quick Overview

Authentication

  • POST /api/accounts/register/ — Register a new user
  • POST /api/accounts/login/ — Log in and obtain JWT tokens

Products

  • GET /api/v1/products/ — List all products
  • GET /api/v1/products/{id}/ — Product details
  • POST /api/v1/products/ — Create (Admin only)

Other Features

  • 📦 Categories
  • ⭐ Reviews
  • 🖼️ Product Images
  • 💖 Wishlists
